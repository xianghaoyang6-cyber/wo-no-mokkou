# 強固點餐 v1.11 圖標內嵌修正版

用途：網站更新用。

本版修正：
- 手機用 content:// 本機預覽 index.html 時，首頁左上小圖標可能讀不到 icons/icon-192.png，導致顯示「強固點餐圖標」文字。
- 首頁小圖標改成內嵌圖片，避免本機預覽破圖。
- App 名稱維持「強固點餐」。
- PWA manifest、favicon、icons 資料夾保留，正式網站安裝圖標仍使用 icons 目錄。
- service worker cache 更新到 v1.11，避免舊快取卡住。

上傳方式：
把本資料夾內所有檔案與 icons 資料夾一起上傳到網站根目錄，覆蓋舊檔。

注意：
如果已安裝到手機桌面，桌面圖標仍顯示舊版時，先移除舊 PWA，再重新加入主畫面。
