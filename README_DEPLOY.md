# 甲一飯包點菜網站部署包 v1.9 預設菜單版

此資料夾可直接部署到 GitHub Pages / Netlify / Cloudflare Pages。

## 版本重點

- 將使用者提供的品項備份檔設為系統預設菜單。
- 預設店名：`甲一飯包`。
- 預設品項：19 個，所有預設數量皆為 0。
- 保留 v1.8 的備份置底、品項備份 / 讀取備份功能。
- 保留已選品項合計、總數量、合計金額、店名編輯、書法品名。
- Service Worker 快取版本：`jiayi-order-site-v1-9`。

## 預設菜單來源

來源備份檔：`甲一飯包_品項備份_20260615_1022.json`

預設菜單只會在沒有本機儲存資料時套用。若手機瀏覽器已經存有舊菜單，更新網站後會優先讀取本機舊資料，不會自動覆蓋。

## 上傳方式

把本資料夾內所有檔案上傳到網站根目錄，不要只上傳 `index.html`。

必要檔案：

```text
index.html
manifest.webmanifest
service-worker.js
404.html
favicon.ico
icons/
```

## 更新提醒

若手機仍顯示舊版，請重新整理；仍不更新時，清除該網站資料後重新開啟。
